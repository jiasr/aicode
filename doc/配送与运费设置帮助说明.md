# 配送与运费设置帮助说明

## 一、配送方式

### 配送方式有哪几种？

系统支持 **3 种配送方式**，每个商品可选择其中一种：

| 配送方式 | 说明 | 适合场景 |
|---------|------|---------|
| **快递** | 用户填写收货地址，商家通过快递发货 | 绝大多数商品 |
| **自提** | 用户到指定门店自取，无需运费 | 有线下门店的商品 |
| **同城配送** | 商家安排同城配送，用户选择时段 | 生鲜、外卖等时效性强的商品 |

> **设置入口：** 商品管理 → 编辑商品 → 配送方式

---

## 二、快递运费模板

### 什么是运费模板？

运费模板用来定义**快递配送时如何计算运费**。例如：

- "江浙沪8元" → 发到江浙沪统一8元
- "首件10元 续件5元" → 买1件10元，每加1件加5元
- "满99元包邮" → 订单满99元免运费

> **设置入口：** 系统设置 → 运费模板

### 运费模板的组成部分

一个运费模板包含：

```
运费模板
  ├── 基础设置（计费方式 + 默认价格）
  ├── 包邮条件（满额包邮）
  └── 地区规则（各省份单独定价，可选）
```

### 计费方式说明

#### 固定运费

每单收取固定金额，**和买多少件无关**。

| 设置项 | 说明 |
|-------|------|
| 固定运费 | 每单收取的运费金额 |

**示例：** 固定运费设置 20 元 → 买1件20元，买10件也是20元

**适合：** 大件商品（如家具、家电），运费固定

---

#### 按件收费

按商品数量累加计算运费，**买得越多运费越高**。

| 设置项 | 说明 | 示例值 |
|-------|------|-------|
| 首件数量 | 收"首费"的件数 | 1 件 |
| 首件费用 | 首件收取的运费 | 10 元（即第1件10元） |
| 续件数量 | 每多少件收一次"续费" | 1 件 |
| 续件费用 | 每续件数量收取的运费 | 5 元（即每加1件加5元） |

**计算示例：**

```
首件1件10元，续件1件5元

买1件 → 10元 = 10元
买3件 → 10 + 5×2 = 20元
买5件 → 10 + 5×4 = 30元
```

**常见变体：**

```
首件1件10元，续件3件6元

买1件 → 10元
买2件 → 10 + 6 = 16元（第2件算1个续件）
买4件 → 10 + 6 = 16元（第2~4件算1个续件）
买5件 → 10 + 6×2 = 22元（第2~4件算1个续件，第5~7件算1个续件）
```

> **提示：** 如果设置了"首件=1，续件=1"，就是"每件X元"的效果

---

### 包邮条件说明

#### 满额包邮

订单金额达到指定门槛后免运费。

| 设置项 | 说明 | 示例 |
|-------|------|------|
| 包邮门槛 | 订单满多少元免运费 | 99 元 |

**示例：** 设置满 99 元包邮

```
订单金额 50元 → 正常收运费
订单金额 120元 → 免运费
```

> **注意：** 满额包邮是在运费计算**之前**判断的，满足条件直接运费为0

---

### 地区规则说明

#### 什么是地区规则？

不同省份的运费可以单独设置，**未设置的省份使用模板默认值**。

| 配置项 | 说明 |
|-------|------|
| 省份 | 选择需要特殊设置的省份 |
| 是否包邮 | 勾选后，该省份运费全免 |
| 固定运费 | 覆盖模板的固定运费（仅固定运费模式） |
| 首费 | 覆盖模板的首件费用（仅按件模式） |
| 续费 | 覆盖模板的续件费用（仅按件模式） |
| 包邮门槛 | 覆盖模板的包邮门槛 |

**使用场景示例：**

```
场景1：偏远地区加价
  模板默认：首件10元，续件5元
  新疆 → 首件15元，续件10元（单独加价）

场景2：部分地区包邮
  模板默认：首件10元
  西藏 → 勾选"包邮"（西藏免运费）

场景3：部分地区降低包邮门槛
  模板默认：满99元包邮
  广东省 → 包邮门槛改为 50 元（广东满50就包邮）
```

> **操作技巧：** 可以选中多个省份，批量设置相同的价格

## 三、完整配置示例

### 示例1：按件 + 满额包邮 + 地区加价

```
模板名称：标准运费
计费方式：按件收费
  首件1件10元，续件1件5元
包邮门槛：满99元包邮

地区规则：
  ┌─────────┬───────────┬────────┬────────┐
  │  省份   │  是否包邮  │  首费  │  续费  │
  ├─────────┼───────────┼────────┼────────┤
  │  新疆   │    否      │  18元  │  10元  │
  │  西藏   │    是      │   —    │   —    │
  │  广东   │    否      │   —    │   —    │ ← 包邮门槛改为 50元
  └─────────┴───────────┴────────┴────────┘

下单效果：
  ★ 3件商品到上海（总价100元）
     → 满99元包邮 → 免运费
  ★ 3件商品到上海（总价50元）
     → 不满99元 → 10 + 5×2 = 20元
  ★ 3件商品到新疆（总价50元）
     → 新疆规则 → 18 + 10×2 = 38元
  ★ 任意到西藏 → 包邮 → 0元
  ★ 3件商品到广东（总价60元）
     → 广东包邮门槛50元，满50 → 免运费
  ★ 3件商品到广东（总价30元）
     → 不满广东包邮门槛 → 10 + 5×2 = 20元
```

### 示例2：固定运费 + 部分地区免邮

```
模板名称：大件运费
计费方式：固定运费
  固定运费：30元
包邮门槛：不启用

地区规则：
  ┌─────────┬───────────┬────────────┐
  │  省份   │  是否包邮  │  固定运费  │
  ├─────────┼───────────┼────────────┤
  │  西藏   │    是      │     —      │
  │  新疆   │    否      │    50元    │
  └─────────┴───────────┴────────────┘

下单效果：
  ★ 送到上海 → 30元
  ★ 送到西藏 → 0元（包邮）
  ★ 送到新疆 → 50元（地区规则覆盖）
```

## 四、常见问题

**Q：一个商品可以同时支持快递和自提吗？**
A：当前版本一个商品只能选择一种配送方式。如果需要多种方式，建议拆分为不同商品（比如"XX商品-快递版"和"XX商品-自提版"）。

**Q：不同商品用了不同运费模板，一起下单怎么算？**
A：取运费最高的模板来计算。例如商品A运费20元，商品B运费10元，一起下单运费按20元收。

**Q：设置了满额包邮，用户部分退款后还包邮吗？**
A：退款后订单金额低于包邮门槛，**不退还已收取的运费**。

**Q：地区规则和模板默认值是什么关系？**
A：模板默认值相当于"通用规则"。地区规则是"特殊规则"，设置了就覆盖默认值，没设置的省份就直接用默认值。
